Sistema de gestion de juegos

Proyecto desarrollado en Python como parte final del Modulo 3

Descripcion:
Aplicacion que permite gestionar un catalogo de juegos, aplicando estructuras de datos, validaciones y estadisticas.

El sistema permite hacer lo siguiente:
-Agregar juegos con ID unica
-Mostrar juegos registrados
-Buscar juegos por ID
-Eliminar juegos
-Visualizar estadisticas (total juegos, precio promedio, juego mas caro y mas barato)

Tecnologias utilizadas:
-Python
-Listas
-Diccionarios
-Conjutos (set)
-Funciones
-Manejo de expeciones (try/except)
